# ☕ Coffee Analytics BI - Dashboard Operativo

### Visión del Proyecto
Plataforma de Business Intelligence diseñada para la optimización de márgenes y eficiencia operativa en cadenas de café premium (Caso de estudio: Culto Café, Uruguay).

## 🚀 Stack Tecnológico
- **Frontend:** Next.js 14 (App Router), Tailwind CSS.
- **Iconografía:** Lucide React.
- **Despliegue:** Vercel (CI/CD integrado).
- **Metodología:** Agile / Scrum.

## 🛠️ Configuración Local
1. Clonar el repositorio.
2. Ejecutar `pnpm install` o `npm install`.
3. Correr el servidor de desarrollo: `npm run dev`.
